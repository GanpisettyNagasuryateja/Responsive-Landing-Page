# Responsive-Landing-Page
<h1>Objective</h1>
Build a clean, responsive landing page with a header, hero section, and footer.
<h1>Procedure</h1>
1.Create index.html and style.css.
    Created an HTML File index.Html.
    Created a CSS file style.css.
2.Setup basic HTML5 structure and link CSS.
    Used the HTML5 (<!DOCTYPE html>, <html>, <head>, <body>).
    Linked CSS Stylesheet using:
    <link rel="stylesheet" href="style.css">
3.Add a header with logo and nav links.
    Created a header contaning:
    Logo(MyLogo)
    Navigation Links:Home,Services,About,Contact
4.Design a hero section with heading, paragraph, and a call-to-action button.
    Created a section with class has hero
    Added Heading "Welcome to our Website"
    Added a paragraph.
    Created a "get started" button as the Call-To-Action.
5.Use CSS Flexbox/Grid for layout.
    Used Flexbox in the header to align the logo and navigation.
    Used CSS Grid in the hero section to center the content.
6.Apply media queries for responsiveness (e.g., nav co lapse on sma l screens).
    Added a media query for screens 768px and below.
7.Style footer with social links.
    Added the footer.
    included social media links:facebook,instagram,X.
    Styled the footer with dark background,padding,center aligment.
8.Open with live-server and test responsiveness by resizing window.
    Opened index.html using the Live Server extension.
    Verified the responsiveness by resizing the browser window and observing the layout changes.
